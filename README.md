# App web para organizar tareas

## Funciones
- Crear tareas
- Marcarlas como completas
- Borrar las tareas
- Las tareas quedarán guardadas aunque se cierre el navegador

## Tecnologías Usadas
- HTML
- CSS
- Javascript
